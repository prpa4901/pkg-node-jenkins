Node.js DevOps Test App

Welcome to our test application!

This is a simple Node.js application created for learning DevOps practices with Jenkins.

API Endpoints

This application provides the following API endpoints:

Root Endpoint

GET /api

Returns a welcome message in JSON format.

Health Check

GET /api/health

Returns the health status of the application.

About This Project

This project demonstrates:

  • Setting up a Node.js application
  • Implementing basic API endpoints
  • Testing with Jest
  • CI/CD with Jenkins
  • Containerization with Docker
